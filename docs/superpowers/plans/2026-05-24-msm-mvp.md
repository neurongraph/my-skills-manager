# MSM MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a working `msm` Python CLI that implements the MVP behavior defined in the project specs.

**Architecture:** The CLI delegates to focused package modules for config, registry, profiles, agents, deployment, state, and orchestration. Config files are authoritative; deployment state is generated metadata.

**Tech Stack:** Python 3.12+, uv, Typer, Rich, PyYAML, pydantic, pytest.
